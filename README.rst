Burrito
=========

Announcement
-------------

We announce the sunset of this project.
We are shutting down the project as of Nov. 10, 2025.

This project, which began in November 2022, will be stopped
and the development will no longer proceed here.
Anyone who wishes to continue this project may fork the project and
continue the development.

We plan to launch a new project.
The new project will be developed on a new architecture based only
on Debian GNU/Linux.

Introduction
------------

Burrito is a security-hardened OpenStack on Kubernetes Platform.

Online documentation can be found at
https://burrito.readthedocs.io/.
